# 痕迹 - 美学设计规范

## 一、设计理念

> "于留白处见天地，于墨痕间见人心。"

整体设计取意中国传统手卷与古籍版式，追求**沉静、克制、留白、余韵**。界面如展开的宣纸，交互如提笔落墨，每一个动作都留下可追溯的"痕迹"。

## 四、核心视觉元素设计规范

### 4.1 宣纸底

```css
.xuan-paper {
  background-color: #F5EFE0;
  background-image: 
    /* 纸纤维纹理 */
    url("data:image/svg+xml,..."),
    /* 淡墨晕染 */
    radial-gradient(ellipse at 20% 30%, rgba(200,64,50,0.03) 0%, transparent 50%),
    radial-gradient(ellipse at 80% 70%, rgba(74,85,104,0.03) 0%, transparent 50%);
  /* 边缘微卷 */
  box-shadow: 
    inset 0 0 60px rgba(139,105,20,0.08),
    inset 0 0 200px rgba(139,105,20,0.04);
}
```

### 4.2 朱砂印章（评分）

- **形状**：正方形（方印），略带不规则边缘模拟手工篆刻
- **色彩**：朱砂红 `#C84032`，略带肌理
- **字法**：印章内刻篆书风格的"上/中上/中/中下/下"对应五星
- **盖印效果**：轻微旋转（±3°随机），透明度 0.85，模拟印泥蘸取不匀
- **交互**：点击印章区域时，先出现一个红色渐隐扩散的"盖印"动画

评分对应：
- 5 分（上）：完整鲜红方印，印文"上上品"
- 4 分（中上）：鲜红方印，印文"上"
- 3 分（中）：朱砂稍淡，印文"中"
- 2 分（中下）：朱砂淡，印文"次"
- 1 分（下）：朱砂极淡，印文"下"
- 未评分：空白方框（仅边框）

### 4.3 旧书页笔记

- **纸张质感**：偏黄的旧纸色 `#F0E6D2`，加细密横线纹理
- **折痕**：左侧或右侧有模拟折痕的阴影
- **墨水感**：文字略带晕染边缘（text-shadow 模拟）
- **批注位置**：支持在正文旁写"眉批"、"夹注"样式
- **装饰元素**：可在角落添加淡墨梅花或兰草小纹样

### 4.4 折扇统计面板

- **形态**：半圆形扇面展开，扇骨为赭石色细线
- **分区**：每一扇面分区展示一个统计维度（数量、时长、类型分布等）
- **扇面纹理**：仿绢布纹理，淡色分区
- **展开动画**：默认收起为一把折扇，点击后平滑展开为半圆形
- **数据呈现**：扇面上以墨点、墨线的疏密表现数据大小

### 4.5 墨点热力图（周统计）

- **基底**：米白宣纸
- **时间轴**：7 列（周一至周日）× 多行（周数）
- **数据点**：
  - 无活动：不着墨（留白）
  - 少量活动：极淡的小墨点（焦墨）
  - 中量活动：中等墨点（淡墨）
  - 大量活动：浓墨大点，周围带微晕染（类似墨滴入水）
- **点缀**：随机在空白处点缀极淡的小墨点，模拟"梅花落纸"
- **交互**：悬停显示当日数据详情，墨点微放大并加深

### 4.6 数字长卷（年度报告）

- **形态**：横向长卷轴，可左右拖动/滑动浏览
- **结构**：
  - 卷首：题字"某某年痕迹"，加落款印章
  - 卷中：分段展示全年数据，每段一幅小景
  - 卷尾：总结语 + 全年关键词
- **视觉**：
  - 以时间为横轴，展开山水手卷的视觉叙事
  - 数据可视化融入山水画意境：读过的书如山峦叠起，看过的剧如江河蜿蜒
  - 每个节点标注作品名与简短感想
- **交互**：
  - 鼠标/触摸左右拖动浏览
  - 滚轮缩放局部
  - 点击节点查看详情

## 五、留白与间距

遵循"计白当黑"原则：

| 元素 | 内边距 | 外边距 |
|------|--------|--------|
| 页面主容器 | 60px 80px | - |
| 卡片/宣纸块 | 40px 48px | 24px 0 |
| 段落间距 | - | 1.5em |
| 标题与正文间距 | - | 0.8em |
| 元素最小留白 | - | 24px |

---

## 六、动效规范

所有动效追求**自然、克制、余韵**：

1. **页面转场**：如徐徐展开手卷，透明度 0→1，同时轻微下→上位移（20px），时长 600ms，缓动 `cubic-bezier(0.25, 0.46, 0.45, 0.94)`
2. **印章盖下**：先放大 120% 透明度 0，再缩至 100% 透明度 0.85，时长 300ms
3. **墨迹晕染**：hover 元素时，背景以 radial-gradient 从中心晕开淡墨色，时长 400ms
4. **折扇展开**：rotate 从 -150° ~ +150° 逐扇骨展开，总时长 800ms
5. **长卷拖动**：惯性滚动 + 回弹，手感柔和

---

## 七、无障碍与响应式

- 正文最小对比度不低于 4.5:1（黛蓝 `#4A5568` 对米色 `#F5EFE0` 对比度约 7:1 ✓）
- 字号不小于 12px（仅批注用）
- 移动端：宣纸边距缩减为 20px，折扇与长卷自适应宽度
- 深色模式（可选）：切换为"绢本"深色系，墨色反转

## 八、设计验收清单

- [ ] 任意页面截图，整体色感不超过 4 种主色
- [ ] 正文阅读舒适度：连续阅读 5 分钟不觉刺眼
- [ ] 所有交互有明确的"落墨"反馈（动效/色彩变化）
- [ ] 留白比例 ≥ 40%（视觉感受）
- [ ] 印章、折扇、长卷等特色元素无违和感
